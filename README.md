### 创建vue3 项目 （Vite方式）
```
Vite 原理是利用Es6 的import 会发送请求去加载文件的特性,拦截这些请求，做些预编译，省去webpack 冗长的打包时间

1.安装 Vite ： npm install -g create-vite-app
2.创建vite vue3 项目 create-vite-app projectName

vue2.0 存在饿问题就说数据和方法分散


```
